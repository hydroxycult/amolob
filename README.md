# Slime Blob

A small terminal program that simulates a soft slime blob using springs, pressure and simple forces.

Build with:

```
cc slime_blob.c -lm -o slime_blob
```

Run with:

```
./slime_blob
```

Controls are available while the slime is running.
Movement forces: w, a, s, d.
Quit with q.
Force modes: 1, 2, 3, 4.
Gravity toggle: g.
Wind: f for right, F for left.
Turbulence: T.
Color themes: c.
Diagonal nudges: X and C.
Radial forces: p (out), i (in), o (pulsing).
Rotation: j, k, l.
Squeeze: h and v.
Stretch: H and V.
Vibration: b and B.
Wave motion: n.
Directional stretch: u and y.

Runs in any terminal with ANSI escape support.
